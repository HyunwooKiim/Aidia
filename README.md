# Team Aidia
## 현재 진행 중인 프로젝트
### 저희가 현재 개발 중인 프로젝트는 사람들이 일상에서 떠올리는 다양한 아이디어를 자유롭게 공유할 수 있는 서비스입니다. 이 서비스에서는 누구나 자신이 떠올린 아이디어를 올리고, 다른 사람들이 그 아이디어를 보고, 필요하다면 자신의 프로젝트에 활용할 수 있습니다. 이를 통해 아이디어가 단순히 사라지는 것이 아니라, 실제로 구현되고 발전할 기회를 제공하고자 합니다.
